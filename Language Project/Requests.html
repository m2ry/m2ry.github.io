Question Requests/Answer a Question:

If you'd like to ask a question or answer one you've come to the right place.


Steps:

  1. Email language.help.4.you@gmail.com

  2. If asking a question, write "Question, language" in the subject. If answering a question, write "Answer, question #, language".

  3. If asking a question, write "your question". If answering a question, write "your answer".


*p.s. CAPITAL LETTERS mean literally, lowercase letters mean specifically.
